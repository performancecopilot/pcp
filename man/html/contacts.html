Contacts


Mailing List

  Questions and discussion about the toolkit should be directed to the PCP mailing list, http://pcp.io/pipermail/pcp.

IRC Channel

  Several PCP developers and fellow users can be contacted via the #pcp channel on the Freenode network.

Web pages

  The PCP web pages at http://pcp.io/ cover PCP, PCP GUI and PCP Glider (the native Windows distribution of PCP).

Binaries

  Pre-built packages for Redhat/SuSE, Debian/Ubuntu, Windows, MacOSX and Solaris can be downloaded from the PCP website at ftp://ftp.pcp.io/projects/pcp/download/.

Source Code

  GIT: git://git.pcp.io/pcp.git.

  GIT web: http://git.pcp.io/cgi-bin/gitweb.cgi?p=pcp.git.

  Compressed source tarballs can be found alongside the binary packages above.
